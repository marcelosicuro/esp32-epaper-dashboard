# Fontes necessárias

Baixe os arquivos abaixo e salve nesta pasta (`fonts/`):

| Arquivo | Link |
|---|---|
| `Roboto-Bold.ttf` | https://fonts.google.com/specimen/Roboto → Download family |
| `Roboto-Regular.ttf` | (mesmo pacote acima) |
| `Roboto-Light.ttf` | (mesmo pacote acima) |

**Atalho via terminal:**

```bash
curl -L "https://fonts.gstatic.com/s/roboto/v47/KFOMCnqEu92Fr1ME5WNLoka5CA.woff2" -o Roboto-Regular.ttf
```

Ou usando o `pip install gfonts` e depois `gfonts download Roboto`.

> As fontes Roboto são licenciadas sob Apache 2.0 — uso livre, incluindo comercial.
